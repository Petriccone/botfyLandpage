# Botfy Landing Page

Landing page da Botfy — infraestrutura de agentes de IA seguros (React + Vite + Tailwind, i18n EN/PT/ES).

## Como rodar

```bash
npm install
npm run dev
```

## Subir para o GitHub

1. Crie um repositório **vazio** no GitHub com o nome **botfyLandpage** (sem README, sem .gitignore).
2. No terminal, na pasta do projeto:

```bash
git remote add origin https://github.com/SEU_USUARIO/botfyLandpage.git
git push -u origin main
```

Substitua `SEU_USUARIO` pelo seu usuário do GitHub.
